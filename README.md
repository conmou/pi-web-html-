## 前端 -> piFronted
#### 連接api，呈現圖表及資料
- Html 呈現
- 圖表使用 Chart.js
- js接api

## 後端 -> piBackend
#### 後端接值，開api 
- express.js 為後端架構
- Firebase 為雲端資料庫

## pi local 資料傳輸 -> pushDataToFirebase
- 使用ＭＱＴＴ協定來溝通
- `sudo service mosquitto start`啟用協定
- pub於樹莓派發布資料
- sub於電腦端接收資料並寫入Firebase
